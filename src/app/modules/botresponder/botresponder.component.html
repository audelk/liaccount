Want to communicate with your existing network? Start new Messenger Campaign.

  • Set up steps with personalized automated messages for your first message or for a follow-up message, set another automated follow-up message if no response is received in a few days. This is done by clicking on the Messages tab inside the Messenger Campaign.

  • To add people from your network to the Messenger Campaign click on Connections in the top menu. Select people you would like to be added to Messenger Campaign(you can select all by click the top most checkbox) and click 'Add selected contacts to Messenger Campaign'. Select contacts by clicking the checkboxes for each contact.
